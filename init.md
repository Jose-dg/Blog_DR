<!-- Existen algunos modificadores en solidity:

    "indexed", se les indica a los nodos de Ethereum que deben indexar estos valores para que puedan ser buscados y filtrados más fácilmente en la blockchain.

    "anonymous": Este modificador se puede usar para ocultar el valor de un parámetro en el evento, lo que significa que no se indexará ni se guardará en la blockchain. Esto puede ser útil si desea ocultar información confidencial, como una dirección de billetera.

    "non-indexed": Este modificador se puede usar para indicar que un parámetro no se debe indexar en la blockchain. Esto puede ser útil si desea guardar información, pero no desea que se indexe para que pueda ser buscada y filtrada fácilmente.

    "memory": Este modificador se puede usar para indicar que un parámetro solo se almacena en la memoria temporalmente y no se guarda en la blockchain. Esto puede ser útil si desea realizar operaciones con datos sin guardarlos en la cadena de bloques.

        Es importante tener en cuenta que el uso de modificadores en los parámetros del evento puede afectar el costo de gas de las transacciones.
